# Directorio de Logos Institucionales (.webp)

Este directorio ha sido creado físicamente por el **Agente de Infraestructura** para alojar los logos estáticos de tu stand.

## Instrucciones para GZ7:
1. Copia tu archivo de imagen del logo de la UNASAM en esta carpeta y renómbralo exactamente como:
   `unasam.webp`
2. Copia tu archivo de imagen de CIBERSEC en esta carpeta y renómbralo exactamente como:
   `cibersec.webp`

El simulador detectará automáticamente los logos una vez cargados en caliente gracias a las directivas fallback reactivas inyectadas en la interfaz de usuario.
